1.2. Save User

Save user to document

Sample Request
POST /api/v1/user/save?document=driver HTTP/1.1
Content-Type: application/json
Accept: application/json
Content-Length: 81
Host: localhost:8080

{
  "name" : "marjuki",
  "email" : "email@marjuki.com",
  "photo" : "photoUrl"
}
param type desc

document

parameter

Document where user saving, driver, passenger or driver_active

user

body / Content-Type: application/json

Put your user object json in body
Sample Response
HTTP/1.1 200 OK
Content-Type: application/json
Content-Length: 272

{
  "message" : "save in `driver` document",
  "data" : {
    "id" : "a86bd35a-5409-399c-8310-6ee6f746382e",
    "name" : "marjuki",
    "email" : "email@marjuki.com",
    "phone" : null,
    "photo" : "photoUrl",
    "position" : null,
    "driver_attribute" : null
  }
}
